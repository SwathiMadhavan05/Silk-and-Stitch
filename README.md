# Fashion Stylist OpenEnv

An OpenEnv-compliant AI environment for fashion styling and Bangalore boutique supply chain optimization.
Now enhanced with PPO RL agent, LSTM demand forecasting, and SHAP explainability.

## Quick Start

    pip install -r requirements.txt
    python app.py

Open http://localhost:7860 for the dashboard.
